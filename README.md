# LeetCode Hot 100 题卡

一个本地可直接打开的 LeetCode Hot 100 学习题卡站点，按题型分组展示题目、难度、例子、一句话思路、图解、Java 答案和个人掌握状态。

## 功能

- 100 道 Hot 100 题卡，按哈希、双指针、滑动窗口、前缀和、单调队列、二叉树、回溯、动态规划等类型分组
- 每题包含题目、例子、一句话思路、简单解释、复杂度和带注释的 Java 答案
- 支持搜索、难度筛选、上一题/下一题、复制代码
- 支持本地记录“已掌握”和个人笔记，数据保存在浏览器 localStorage
- 支持桌面端和手机端浏览

## 使用方式

直接用浏览器打开：

```text
index.html
```

也可以在当前目录启动一个简单静态服务：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 项目结构

```text
.
├── index.html          # 页面结构
├── styles.css          # 页面样式和响应式适配
├── app.js              # 交互逻辑、搜索筛选、题卡渲染
├── cards.js            # Hot 100 题卡数据和 Java 答案
└── diagrams/           # 定制算法图解
```

## 说明

这个项目是纯静态站点，不依赖打包工具。题卡内容主要服务于复习和理解，Java 代码以清晰、可读、容易复现为优先。
